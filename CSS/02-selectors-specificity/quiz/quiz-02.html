Which selector will target the link to the PDF version?

<a href="http://fto.com/cheatsheet.pdf">Cheat sheet (pdf)</a>
<a href="http://fto.com/cheatsheet.html">Cheat sheet (html)</a>
A. a[href$=".pdf"]
B. a[href^=".pdf"]
C. a[href=endsWith(".pdf")]
D. None of them will target the PDF version
And the answer is...

Answer A
